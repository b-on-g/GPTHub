# GPTHub

Единое окно для всех ИИ-задач — универсальное веб-приложение, объединяющее текст, голос, изображения и работу с файлами в одном чате.

## Запуск (Docker)

```bash
docker-compose up --build
```

Приложение будет доступно по адресу: http://localhost:9080/bog/GPTHub/app/-/test.html

## Запуск (локально)

```bash
# Из корня MAM-монорепы
npx mam bog/GPTHub
npm start
```

Приложение будет доступно по адресу: http://localhost:9080/bog/GPTHub/app/-/test.html
